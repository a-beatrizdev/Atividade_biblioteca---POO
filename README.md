# Sistema de Gestão de Biblioteca (POO)

## Descrição
Sistema simples de gerenciamento de itens de biblioteca desenvolvido em Python utilizando Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de Livros
- Cadastro de Revistas
- Listagem de Itens (polimorfismo)
- Empréstimo de Itens
- Devolução de Itens
- Encapsulamento com getters e setters
- Herança entre classes
- Sobrescrita de métodos

## Como executar

1. Clone o repositório:
git clone https://github.com/a-beatrizdev/Atividade_biblioteca---POO

2. Acesse a pasta:
cd biblioteca-poo

3. Execute:
python main.py

## Exemplo de uso
1 → Cadastrar Livro  
3 → Listar Itens  
4 → Emprestar Item  
5 → Devolver Item
